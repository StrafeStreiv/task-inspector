# Инспектор задач

Мини-сервис для управления задачами с веб-интерфейсом. Реализован на ASP.NET Core 7 + EF Core + SQLite.

## Функциональность

- Создание, просмотр, изменение статуса задач.
- Комментирование задач.
- Фильтрация списка по статусу и исполнителю.
- Аналитика: общая статистика, просроченные задачи, среднее время выполнения.
- Бизнес-правила:
  - Название от 5 до 200 символов.
  - Срок не может быть в прошлом.
  - Критические задачи без срока получают дедлайн +24ч.
  - Завершённые задачи нельзя изменить.
  - У одного исполнителя не более 3 задач в работе.
  - Только последовательные переходы Open → InProgress → Completed.
  - Просроченность вычисляется динамически.

## Технологии

- **Backend**: ASP.NET Core 7, Entity Framework Core 7, SQLite, Swagger.
- **Frontend**: HTML, CSS, Bootstrap 5, чистый JavaScript.
- **Тесты**: xUnit.

## Запуск

1. Установите .NET 7 SDK.
2. Склонируйте репозиторий.
3. В папке `TaskInspector` выполните:
   ```bash
   dotnet ef database update
   dotnet run
   ```
4. Откройте https://localhost:5001 (или http://localhost:5000).

## Тестирование

Юнит-тесты находятся в проекте TaskInspector.Tests. Запустите:
  ```bash
  dotnet test
  ```

## API документация

После запуска доступен Swagger: /swagger.

## Архитектурные решения

- Слой сервисов инкапсулирует всю бизнес-логику.

- Глобальный middleware обрабатывает ошибки и логирует их.

- Маппинг выполняется вручную (для простоты).

- Конфигурация исполнителей вынесена в appsettings.json.

- Фронтенд — SPA на чистом JS, общается с API через fetch.
